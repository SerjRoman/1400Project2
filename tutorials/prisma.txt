1. npm install prisma
2. Инициализации призмы в проект (npx prisma init)
3. По итогу инициализации создасться .env и папка призмы
4. обязательно добавить .env в .gitignore
5. .prisma - файлы призмы, призма состоит из двух технологий:
    5.1 модели это призма файлы в которых вы указываете связи, поля для модели
    5.2 клиент (призма клиент) это специальный класс в js|ts который позволяет отправлять запросы в БД
    5.3 После изменение модели мы всегда генерируем нового клиента
    5.4 в .env файле указываем путь к БД (если sqlite относительно файла schema.prisma: file:./..., если это любая серверная БД - ссылку на сервер)
    5.5 в schema.prisma файле datasource db в поле provider указываем sqlite 
6. в призма есть аттрибуты и функции. Аттрибуты пишутся с @ или @@, а функции просто выполняют какие то штуки(now() - получить время сейчас, autoincrement() - автоматически +1 к числу)
7. структура любого поля в БД: name Type @attributes
8. Команды по работе с призмой:
    8.1 миграции: npx prisma migrate dev
    8.2 обязательно в гит игнор migrations, generated и .db
    8.3 команда для генерации клиента: npx prisma generate
    8.4 форматируем призма файл npx prisma format
9. Связи в БД
